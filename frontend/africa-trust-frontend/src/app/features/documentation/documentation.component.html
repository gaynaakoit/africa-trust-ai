Comment intégrer nos services

Chaque service peut être intégré via notre API. Les guides ci-dessous expliquent comment obtenir une clé API, envoyer vos requêtes et recevoir les résultats.

Deepfake Detection API

Exemple d'appel API pour analyser une vidéo :

                POST /api/deepfake
                Headers:
                Authorization: Bearer YOUR_API_KEY
                Body:
                {{ '{' }}
                    "video_url": "https://example.com/video.mp4"
                {{ '}' }}

Liveness / Anti-Spoof API

Exemple d'appel API pour vérifier la preuve de vie :

                POST /api/liveness
                Headers:
                  Authorization: Bearer YOUR_API_KEY
                Body:
                  {{ '{' }}
                    "image": "base64_encoded_image_here"
                  {{ '}' }}
